# 清濯洗衣液官网

清濯洗衣液官网是一个用于展示茶皂素复配洗衣液品牌形象、产品卖点、配方灵感与课程团队成果的前端项目。页面以“清洁实验室 × 东方青瓷感”为视觉方向，强调洁净、温和、植物灵感与专业配方背书。

## 项目定位

- **品牌名称**：清濯 QINGZHUO
- **产品类型**：茶皂素复配深层洁净洗衣液
- **展示重点**：产品包装、核心卖点、配方体系、真实使用场景、团队信息
- **适用场景**：课程作品展示、品牌官网原型、产品介绍页面

## 主要内容

- 首屏品牌展示与产品口号
- 产品系列与包装视觉展示
- 深层洁净、浓缩省量、护色护纤、持久留香等卖点展示
- 茶皂素复配配方与实验室素材展示
- 家庭洗护使用场景展示
- 团队与课程背景信息

## 技术栈

- React 19
- TypeScript
- Vite
- Tailwind CSS
- lucide-react
- class-variance-authority

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

构建生产版本：

```bash
npm run build
```

预览生产构建：

```bash
npm run preview
```

## 设计方向

当前版本偏向清爽、克制的品牌展示风格。后续可继续围绕以下方向优化：

- 强化首屏产品瓶身主视觉
- 统一图片色调与裁切比例
- 将产品区和卖点区改为更具画册感的排版
- 优化中文字体、动效节奏和移动端细节
- 压缩图片资源，提升加载性能
